📖 Overview

AirNet Company Network Design & Security is a professional enterprise networking project developed to design a secure, scalable, and centralized network infrastructure for a rapidly growing multinational organization.

The project simulates a real-world corporate network connecting multiple branches across:

🇦🇺 Sydney (Headquarters)
🇦🇺 Melbourne
🇦🇺 Brisbane
🇭🇰 Hong Kong

The infrastructure was designed using advanced networking concepts including:

✅ Variable Length Subnet Masking (VLSM)
✅ VLAN Segmentation
✅ Inter-VLAN Routing
✅ VPN Connectivity
✅ Wireless Network Security
✅ Enterprise Security Controls
✅ Hierarchical Network Architecture

🔀 VLAN Segmentation Strategy

To improve security, traffic management, and departmental isolation, VLAN segmentation was implemented throughout the enterprise network.

📋 VLAN Structure
VLAN ID	Department	Purpose
VLAN 10	Management	Administrative Access
VLAN 20	Human Resources	HR Department
VLAN 30	Finance	Financial Operations
VLAN 40	IT Support	Technical Support
VLAN 50	Guest Network	Restricted Guest Access
VLAN 60	Server VLAN	Centralized Enterprise Servers
🖥️ Dedicated Server Infrastructure

A dedicated Server VLAN (VLAN 60) was created on Level 5 of every branch office.

🔒 Security Benefits
Controlled access policies

Enhanced firewall monitoring
Reduced attack surface
Improved performance and management
